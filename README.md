# WIP: acot-reporter-github

A GitHub reporter for [acot](https://github.com/acot-a11y/acot).

## Usage

### Installation

Install via npm:

```bash
$ npm install --save acot-reporter-github
```

Install via yarn:

```bash
$ yarn add acot-reporter-github
```

### Configuration

```javascript
module.exports = {
  reporter: "github",
};
```
